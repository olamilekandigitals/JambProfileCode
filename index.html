Generate JAMB Profile Code

How It Works

  • You will be redirected to your phone's SMS app.
  • The message will be pre-filled automatically.
  • All you need to do is tap Send.

Important Notes

  • Airtime: Ensure you have at least ₦50 airtime.
  • SIM Card: Use the phone number linked to your NIN.
  • Old Candidates: You may reuse your old profile code.

Profile Code Retrieval

To retrieve your profile code, send RESEND to 55019 or 66019 from the registered phone number previously used to generate the profile code.

First-Time / Lost Code

Text NIN followed by a space and your 11-digit NIN to 55019 or 66019.
Example: NIN 12345678901

Note: A small fee (around ₦50) is usually deducted for this service.

For Previous Year's Candidates

Your profile code remains the same. Use the RESEND method or retrieve it from the JAMB portal.

Key Reminder: The profile code is a unique 10-digit number that links all your JAMB activities.

When Will I Get My Profile Code?

You will usually receive your profile code instantly or within a few minutes. During peak periods, delays may occur and can take up to 15 minutes or longer. Please be patient.

If You Don’t Get It

  • Wait a few minutes due to possible network congestion.
  • Resend the SMS if there is no response after some time.
  • Ensure you used the phone number linked to your NIN.
